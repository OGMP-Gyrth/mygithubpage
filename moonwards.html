Moonwards

I was fortunate enough to be part of a the Moonwards team.
This project aims to create a realistic colony on the moon.

It is made with the Godot Engine and available under the MIT License.

Character Control

A big part of my job was scripting the character interaction with the environment. This included character movement in low gravity and using the vertical stairs designed for low gravity.

These features were also connect via the networking API to make character sync over the network.

Tours

A large part of the project is about education. This is done by audio tours and interactive displays such as the Bucknell rocket. The user can inspect parts of the rocket and learn how they operate.

To travel long distances an air-tight traveling pod can be used. These can transport a lot of people at once by being lifted up by autonomous rovers.

Environment

The world of Moonwards is massive. To strike a good balance between performance and graphical fidelity we had to use a lot of different techniques. Such as custom shaders, LODs and particle detail objects.
